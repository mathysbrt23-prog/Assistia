# Documents Assistia

Ce dossier contient uniquement les documents utiles au projet actuel.

## Document maitre

- `../LECTURE.docx` : fichier Word de lecture qui regroupe les documents humains importants du projet.

## Product

- `product/assistia-reply-analyse-produit.docx` : analyse produit de la nouvelle piste Assistia Reply, avec faisabilité, MVP, roadmap, risques et pricing.
- `product/rapport-concurrence-assistia-reply.md` : version Markdown de travail du rapport concurrence. La version lisible Word est incluse dans `../LECTURE.docx`.

## Investor

- `investor/assistia-reply-pitch-investisseur.docx` : pitch investisseur complet pour Assistia Reply, avec marché, concurrence, business model, projections financières, ask et annexes VC.

## Produit actuel

Le dossier est maintenant centré sur Assistia Reply :

- extension Chrome pour Gmail et WhatsApp Web ;
- API `/api/reply/generate` ;
- API `/api/extension/token` pour connecter l'extension au compte utilisateur ;
- dashboard d'usage et de préférences ;
- Stripe pour les plans Pro et Business ;
- Supabase pour les profils, historiques légers et quotas.

Les anciens pitchs et modules liés à l'idée d'agent WhatsApp complet ont été retirés du dossier pour éviter la confusion avec la nouvelle direction produit.
