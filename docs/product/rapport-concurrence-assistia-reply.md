# Rapport concurrence - Assistia Reply

Derniere verification : 28 avril 2026

## Resume executif

Oui, il y a deja des concurrents dans cette niche. L'idee "assistant qui genere ou reformule des reponses directement la ou l'utilisateur ecrit" existe sous plusieurs formes :

- assistants natifs des grandes plateformes : Google Gemini dans Gmail, Microsoft Copilot dans Outlook, Apple Intelligence sur iPhone/Mac ;
- assistants d'ecriture generalistes : Grammarly, Wordtune, QuillBot, HyperWrite, Compose AI ;
- assistants email specialises : MailMaestro, Ellie, Remail, Superhuman Mail, Shortwave, Canary Mail, Spark, Missive ;
- assistants de messagerie/WhatsApp Web : Draftly, ReplyWizard, Smart Reply Pro, Smart AI Reply, WAPlugin, WeReplyX ;
- claviers mobiles IA : ParagraphAI, Grammarly Keyboard, ReplyContext, AI TypePro, SwiftKey avec Copilot.

Conclusion : le besoin est valide, mais "generer des reponses IA" seul n'est pas assez differenciant. Pour que Assistia fonctionne, il faut viser un angle plus precis : reponses professionnelles en francais, tres simples, integrees a Gmail + WhatsApp Web, avec des cas d'usage concrets comme relances client, refus polis, negociation prix, suivi commercial, excuses, disponibilites, paiement, SAV.

## Lecture du marche

Le marche se coupe en 5 familles.

1. Les geants integrent l'IA directement dans leurs apps.
   Google, Microsoft et Apple reduisent la valeur d'une extension generique. Si l'utilisateur a deja Gemini, Copilot ou Apple Intelligence, il a deja une partie de la promesse.

2. Les assistants d'ecriture horizontaux sont tres forts.
   Grammarly, Wordtune, QuillBot et Compose AI fonctionnent dans beaucoup de champs texte. Leur avantage est la distribution. Leur faiblesse est qu'ils sont souvent trop generiques.

3. Les assistants email specialises sont plus proches de notre idee.
   MailMaestro, Ellie ou Remail savent repondre a un email depuis Gmail/Outlook. Certains vont plus loin avec auto-drafts, knowledge base, style personnel.

4. Les clients email IA remplacent l'app de mail.
   Superhuman, Shortwave, Canary, Spark et Missive demandent a l'utilisateur de changer d'interface. C'est puissant, mais c'est une friction que Assistia peut eviter.

5. Les extensions WhatsApp Web existent deja.
   Draftly et ReplyWizard sont tres proches de l'idee Assistia Reply cote WhatsApp Web. Cela confirme le besoin, mais montre aussi que l'execution doit etre tres differenciante.

## Comment ils integrent l'agent aux conversations

### 1. Integration native

Exemples : Gemini dans Gmail, Copilot dans Outlook, Apple Intelligence.

L'IA est dans l'interface officielle. L'utilisateur voit une icone ou un bouton dans la fenetre de composition, le thread ou le clavier systeme.

Avantages :

- confiance forte ;
- acces direct au contexte ;
- moins de friction ;
- mobile + desktop couverts.

Faiblesses :

- souvent lie a une suite payante ;
- peu personnalise pour un cas metier precis ;
- ton parfois generique ;
- l'utilisateur reste dans l'ecosysteme Google/Microsoft/Apple.

### 2. Extension navigateur universelle

Exemples : Grammarly, Compose AI, Wordtune, HyperWrite, Smart Reply Pro, ReplyWizard.

L'extension injecte un bouton, une bulle ou un raccourci dans les champs texte. Elle lit souvent le texte selectionne, le champ actif, ou une partie visible de la page.

Avantages :

- ne demande pas de changer d'application ;
- fonctionne sur Gmail, LinkedIn, Slack, WhatsApp Web, etc. ;
- MVP relativement rapide.

Faiblesses :

- depend du DOM des apps web ;
- permissions Chrome sensibles ;
- risque de bugs si Gmail/WhatsApp changent leur interface ;
- beaucoup de concurrents.

### 3. Add-in email specialise

Exemples : MailMaestro, Ellie.

L'outil s'installe dans Gmail/Outlook ou se connecte au compte email. Il peut lire le thread plus proprement et parfois creer des brouillons automatiquement.

Avantages :

- meilleure integration email ;
- contexte email complet ;
- workflows pro : reply, summarize, schedule.

Faiblesses :

- OAuth plus sensible ;
- review Google/Microsoft plus lourde ;
- moins pertinent pour WhatsApp/LinkedIn.

### 4. Client email complet

Exemples : Superhuman, Shortwave, Canary, Spark, Missive.

Ils remplacent l'app email. L'IA devient une fonction de l'inbox.

Avantages :

- controle complet de l'experience ;
- UX plus stable ;
- possibilite d'agents plus avances.

Faiblesses :

- demande de migrer d'outil ;
- adoption plus difficile ;
- concurrence avec des produits tres matures.

### 5. Clavier mobile IA

Exemples : ParagraphAI, Grammarly Keyboard, ReplyContext, AI TypePro, SwiftKey.

Le clavier remplace ou complete le clavier natif. L'utilisateur peut reformuler, traduire, corriger ou generer une reponse depuis n'importe quelle app.

Avantages :

- vrai "partout sur mobile" ;
- compatible avec WhatsApp, Messages, Instagram, LinkedIn, Mail ;
- tres proche de la vision long terme d'Assistia.

Faiblesses :

- onboarding difficile, surtout sur iOS ;
- confiance utilisateur faible : donner acces a un clavier est sensible ;
- Apple impose des limites aux claviers tiers ;
- cout de maintenance mobile plus eleve.

## Analyse detaillee par concurrent

### Google Gemini in Gmail

Gemini est probablement le concurrent le plus dangereux sur Gmail. Google propose deja les fonctions clefs : resumer des threads, composer/repondre avec prompt, polir un brouillon, contextual smart reply, recherche dans l'inbox et reference a des fichiers Drive.

Integration : native dans Gmail desktop et mobile.

Ce que cela signifie pour Assistia :

- impossible de battre Google sur l'integration Gmail pure ;
- Assistia doit gagner sur la specialisation : ton francais, cas d'usage pro, WhatsApp Web/LinkedIn en plus, templates de reponse concrets.

Source : [Gemini in Gmail](https://workspace.google.com/intl/en/products/gmail/ai/)

### Microsoft Copilot Outlook

Copilot permet de creer un brouillon dans Outlook, changer longueur/ton, modifier une selection, resumer un fil et travailler sur mobile. Il est tres fort pour les entreprises deja dans Microsoft 365.

Integration : native Outlook web, desktop, Mac, iOS, Android.

Ce que cela signifie pour Assistia :

- Outlook n'est pas le meilleur point d'entree pour un MVP ;
- si Assistia va sur Outlook, il faudra un angle "plus simple et moins enterprise" ou "reponses commerciales francaises".

Sources : [Copilot Outlook Support](https://support.microsoft.com/en-us/office/draft-an-email-message-with-copilot-in-outlook-3eb1d053-89b8-491c-8a6e-746015238d9b), [Microsoft 365 Copilot Pricing](https://www.microsoft.com/en-us/microsoft-365-copilot/pricing-new)

### Apple Intelligence

Apple integre les Writing Tools au niveau systeme : rewrite, proofread, summarize dans Mail, Notes, Pages et apps tierces. Cela couvre directement une partie de la vision mobile.

Integration : systeme iOS/macOS, accessible depuis le clavier ou la selection de texte.

Ce que cela signifie pour Assistia :

- un clavier mobile Assistia sera dur a justifier sur iPhone si la promesse est seulement "reformuler" ;
- il faut de la valeur metier : contexte client, ton de marque, raccourcis de scenarios, historique, modeles.

Sources : [Apple Support - Writing Tools](https://support.apple.com/guide/iphone/use-writing-tools-iph6f08da1d2/ios), [Apple Newsroom](https://www.apple.com/cm/newsroom/2024/06/introducing-apple-intelligence-for-iphone-ipad-and-mac/)

### Grammarly / Superhuman Go

Grammarly est le concurrent horizontal le plus solide. Il fonctionne dans Gmail, Outlook, Slack, LinkedIn, WhatsApp et beaucoup d'autres apps. Il corrige, reecrit, ajuste le ton, genere du texte et dispose aussi d'un clavier mobile.

Integration : extension navigateur, app desktop, mobile keyboard, integrations larges.

Ce que cela signifie pour Assistia :

- Assistia ne doit pas se positionner comme "un Grammarly francais" ;
- il faut etre plus precis : "la meilleure reponse pro a envoyer maintenant", pas "ameliore ton ecriture partout".

Sources : [Grammarly for Gmail](https://www.grammarly.com/gmail), [Superhuman Suite Pricing](https://superhuman.com/plans)

### MailMaestro

MailMaestro est tres proche du besoin email professionnel. Il propose compose, reply, improve, summarize, AI personality, meeting scheduling, et fonctionne sur Outlook/Gmail.

Integration : add-in email / assistant dans inbox.

Prix : Free avec 3 requetes/semaine ; 12 $/mois en annuel ou 15 $/mois mensuel pour Professional/Team ; Duo 25 $/mois.

Ce que cela signifie pour Assistia :

- MailMaestro est un concurrent direct pour Gmail/Outlook ;
- Assistia peut etre plus leger, plus mobile/conversationnel, et plus focalise sur le francais et WhatsApp Web.

Source : [MailMaestro Pricing](https://www.maestrolabs.com/pricing)

### Ellie

Ellie est un concurrent direct sur le principe "reponse email qui imite ton style". Le plan Casual fonctionne par extension dans Gmail, Outlook Web et Fastmail. Les plans Business/Professional peuvent se connecter directement a Gmail/Google Workspace et Outlook, avec auto-draft et auto-categorise. Ellie insiste sur le fait qu'elle ne send pas sans verification humaine.

Integration : extension + integration inbox selon plan.

Prix : la page analysee ne donne pas clairement les montants des plans principaux, mais mentionne des utilisateurs additionnels a partir de 5 $/user.

Ce que cela signifie pour Assistia :

- la promesse "mimic your tone" existe deja ;
- Assistia doit eviter d'etre juste Ellie en francais, et ajouter WhatsApp/LinkedIn + scenarios metiers.

Source : [Ellie Pricing](https://ellieai.com/pricing)

### Remail

Remail est un assistant Gmail/AI toolkit. Il permet de definir le ton de reponse, supporter 40 langues, detecter la langue de l'email et generer une reponse.

Integration : extension Gmail / assistant web.

Prix : Unlimited 14 $/mois en annuel ou 19 $/mois mensuel ; Team+ 159 $/mois en annuel.

Ce que cela signifie pour Assistia :

- concurrence directe sur Gmail ;
- differenciation possible par UX premium, francais natif, WhatsApp Web et pricing plus simple.

Source : [Remail Pricing](https://www.remail.ai/pricing-remail)

### Compose AI

Compose AI est un assistant Chrome horizontal : autocomplete partout, generation avec `//`, reponses Gmail en un clic, rephrase, ton plus professionnel/amical. Il revendique 300 000 utilisateurs sur Chrome Web Store.

Integration : extension Chrome dans les champs texte.

Prix : free forever + premium, mais le site officiel ne rend pas le prix clair dans la page analysee.

Ce que cela signifie pour Assistia :

- l'autocomplete partout est deja couvert ;
- Assistia doit se concentrer sur les reponses utiles, pas sur finir les phrases.

Sources : [Compose AI](https://www.compose.ai/), [Chrome Web Store](https://chromewebstore.google.com/detail/compose-ai-ai-powered-wri/ddlbpiadoechcolndfeaonajmngmhblj)

### Superhuman Mail

Superhuman est un client email premium. Il cible les utilisateurs tres intensifs, souvent sales/fondateurs. Les plans recents ajoutent AI features, auto drafts, Ask AI, custom auto labels, CRM integrations.

Integration : client email complet, pas extension dans Gmail.

Prix : Starter 30 $/mois ou 300 $/an ; Business 40 $/mois ou 396 $/an ; Enterprise custom.

Ce que cela signifie pour Assistia :

- Superhuman vend la productivite email a prix premium ;
- Assistia peut etre une alternative beaucoup plus legere sans migration d'inbox.

Source : [Superhuman Pricing Plans](https://help.superhuman.com/hc/en-us/articles/38456109456147-Pricing-Plans)

### Shortwave

Shortwave est un email client IA avec agent. Il peut organiser l'inbox, filtrer avec prompts, ecrire/ameliore des emails, apprendre la voix, rechercher et planifier. Il est disponible sur iOS, Android, Mac, Windows.

Integration : client email complet.

Prix : free disponible ; les prix payants doivent etre verifies au moment de souscription.

Ce que cela signifie pour Assistia :

- Shortwave couvre une vision "agent email" tres large ;
- Assistia devrait rester plus petit, plus direct, plus facile a adopter.

Source : [Shortwave](https://www.shortwave.com/)

## Menaces principales

### Menace 1 : fonctionnalite commoditisee

Generer une reponse IA est devenu une fonctionnalite basique. Beaucoup d'extensions le font deja.

Reponse :

- ne pas vendre "AI reply generator" ;
- vendre un resultat concret : "reponds a un client sans paraitre froid", "relance un paiement sans malaise", "refuse poliment", "negocie sans perdre la relation".

### Menace 2 : Google/Microsoft/Apple integrent tout

Les plateformes natives gagnent sur Gmail, Outlook et mobile.

Reponse :

- ne pas se battre sur Gmail uniquement ;
- etre multi-conversation : Gmail + WhatsApp Web + LinkedIn + Slack ;
- proposer des scenarios metiers que les IA natives ne specialisent pas.

### Menace 3 : prix tres bas

ReplyWizard est a 1,99 $/mois et Smart AI Reply annonce 4,99 $/mois.

Reponse :

- eviter un produit trop generique ;
- vendre a 7-9 €/mois si la valeur est claire ;
- proposer un plan gratuit mais limiter intelligemment.

### Menace 4 : confiance et donnees sensibles

Lire emails/messages est tres sensible.

Reponse :

- zero auto-send ;
- zero stockage par defaut du contenu ;
- "lit uniquement le message visible/selectionne" ;
- transparence dans l'interface ;
- mode local/BYOK possible plus tard.

## Opportunite pour Assistia

Il reste une opportunite, mais pas avec une promesse large.

La meilleure niche :

> Assistant de reponses professionnelles en francais, integre a Gmail et WhatsApp Web, pour freelances, commerciaux et petites equipes qui veulent repondre vite sans perdre le bon ton.

Pourquoi cette niche est meilleure :

- les concurrents generalistes parlent surtout anglais ;
- les reponses en francais pro ont beaucoup de subtilite : politesse, distance, fermete, relance, formulation commerciale ;
- WhatsApp est tres utilise pour clients/freelances/artisans/commerces ;
- beaucoup de produits WhatsApp sont orientes automation/CRM, pas "bien repondre humainement" ;
- le prix peut etre bas et simple.

## Differenciation recommandee

Assistia doit eviter :

- "une extension IA qui repond partout" ;
- "un Grammarly moins connu" ;
- "un bot WhatsApp" ;
- "un agent autonome".

Assistia doit pousser :

- francais professionnel natif ;
- reponses courtes, naturelles, non robotiques ;
- pas d'envoi automatique ;
- cas d'usage prets a l'emploi ;
- Gmail + WhatsApp Web d'abord ;
- extension ultra discrete ;
- personnalisation simple du ton ;
- templates metiers : freelance, sales, client support, recrutement, immobilier, e-commerce local.

Exemples de boutons utiles :

- Relancer sans paraitre insistant
- Refuser poliment
- Demander une confirmation
- Repondre a une objection prix
- Transformer mes notes en email pro
- Faire plus court
- Faire plus chaleureux
- Dire non fermement
- Repondre en 3 phrases
- Traduire/repondre en anglais pro

## Recommandation pricing Assistia

Le marche montre deux zones :

- tres bas prix : 1,99 a 4,99 $/mois pour extensions simples ;
- prix pro : 12 a 25 $/mois pour assistants email plus complets ;
- clients email premium : 30 a 40 $/mois.

Pricing conseille pour Assistia MVP :

| Plan | Prix | Contenu |
| --- | --- | --- |
| Free | 0 €/mois | 3 reponses/jour, Gmail, tous les tons, blocage 24h quand le quota est atteint |
| Essential | 4,99 €/mois | 40 reponses/jour, Gmail, templates metiers, preferences de ton |
| Pro | 19,99 €/mois | 200 reponses/jour, Gmail, usage intensif sales/support, historique d'usage sans contenu sensible |

Pourquoi :

- 4,99 €/mois garde un achat impulsif face aux extensions simples ;
- 19,99 €/mois reste coherent avec les assistants email pro plus complets ;
- les quotas quotidiens sont plus lisibles que des quotas mensuels et limitent mieux les abus.

## MVP recommande apres cette veille

### Phase 1 : Gmail + WhatsApp Web

Objectif : prouver que les utilisateurs generent des reponses plusieurs fois par semaine.

Fonctions :

- extension Chrome ;
- bouton Assistia dans Gmail et WhatsApp Web ;
- lecture du texte selectionne ou des derniers messages visibles ;
- generation de 3 variantes ;
- reformulation d'une intention brute ;
- insertion dans le champ texte, jamais envoi automatique ;
- tons : Pro, Direct, Chaleureux, Ferme, Court ;
- templates : relance client, devis, paiement, refus, disponibilite.

### Phase 2 : LinkedIn

Pourquoi LinkedIn :

- cas d'usage commerciaux ;
- reponses a prospects ;
- messages courts ;
- douleur forte pour non-natifs ou profils peu commerciaux.

### Phase 3 : mobile

Ne pas commencer par mobile.

Options plus tard :

- clavier mobile iOS/Android ;
- application mobile avec share sheet ;
- mode "copier une conversation -> Assistia propose une reponse".

Le clavier est puissant mais plus risqué : onboarding, confiance, contraintes iOS, maintenance.

## Verdict final

La niche est concurrentielle, mais pas morte.

La mauvaise version du produit :

> Encore une extension IA qui ecrit des reponses partout.

La bonne version :

> Le copilote de reponses professionnelles en francais pour emails et conversations client, sans changer d'app et sans auto-send.

Le point cle :

Assistia ne doit pas vendre de l'IA. Assistia doit vendre du soulagement dans les moments ou l'utilisateur ne sait pas comment formuler une reponse.
